String::MMM
==========

Perl module for matching strings MasterMind style (Match MasterMind), now available at http://search.cpan.org/~jmerelo/String-MMM/

## INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install
